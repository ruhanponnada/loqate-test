### loqate_installer_aix

| Platform | Version       | Build Compiler                               | Library Dependencies                                                                                                                                                               |
| -------- | ------------- | -------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| AIX      | 5.3 and above | IBM XL C/C++ Enterprise Edition V8.0 for AIX | libC.a(shr_64.o)<br/>libC.a(shr2_64.o)<br/>libC.a(ansi_64.o)<br/>libc.a(shr_64.o)<br/>libC.a(ansicore_64.o)<br/>libc_r.a(shr_64.o) <br/>libC.a(shrcore_64.o)<br/>libC.a(shr3_64.o) |
