### Installed API Reference

The Loqate API is a class-based API designed to be easy and intuitive to use in todayâ€™s preferred class-based languages, as well as easy to wrap into a web service. When calling from Java the classes can be found in the com.loqate package. When calling from .Net the classes can be found in the com.loqate namespace

**NOTE** Please ensure that the following characters are not in the addresses you send to the Loqate engine:_~ (tilde), % (percentage), and _ (asterisk)\*
