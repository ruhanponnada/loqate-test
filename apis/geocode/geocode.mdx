#### DistancesAndDirections Interactive Directions (v2)

Returns the directions between two or more points.

[View the API](/apis/geocode/distancesanddirections-interactive-directions-v2/)

#### DistancesAndDirections Interactive Distance (v1)

Returns the distance between two or more points.

[View the API](/apis/geocode/distancesanddirections-interactive-distance-v1/)

#### Geocoding International Geocode (v1.1)

Returns the WGS84 latitude and longitude for the given location. Supports most
international locations.

[View the API](/apis/geocode/geocoding-international-geocode-v1.1/)

#### Geocoding International PositionToCountry (v1)

Returns the country based on the WGS84 latitude and longitude supplied. No
result is returned if the coordinates are in international waters.

[View the API](/apis/geocode/geocoding-international-positiontocountry-v1/)

#### Geocoding International RetrieveNearestPlaces (v1)

Calculates the nearest places of interest of a given category to a given
location.

[View the API](/apis/geocode/geocoding-international-retrievenearestplaces-v1/)

#### Geocoding International ReverseGeocode (v2)

Returns the nearest address or location to the given coordinates.

[View the API](/apis/geocode/geocoding-international-reversegeocode-v2/)

#### Geocoding UK Find (v2)

Finds locations matching the given input. Supports UK only.

[View the API](/apis/geocode/geocoding-uk-find-v2/)

#### Geocoding UK Geocode (v2.1)

Returns the OS easting + northing along with WGS84 latitude and longitude for
the given postcode. Supports UK only.

[View the API](/apis/geocode/geocoding-uk-geocode-v2.1/)

#### Geocoding UK Retrieve (v2)

Returns the OS easting + northing along with WGS84 latitude and longitude for
the given location. Supports UK only.

[View the API](/apis/geocode/geocoding-uk-retrieve-v2/)

#### Geocoding UK RetrieveNearestPlaces (v1.2)

Calculates the nearest places of interest of a given category to a given
location.

[View the API](/apis/geocode/geocoding-uk-retrievenearestplaces-v1.2/)

#### Geocoding UK ReverseGeocode (v1.1)

Returns the nearest address or location to the given coordinates. Currently only
supports the UK.

[View the API](/apis/geocode/geocoding-uk-reversegeocode-v1.1/)

#### GovernmentData Postzon RetrieveByCoordinates (v1.2)

Returns the Postzon record nearest to the given coordinates.

[View the API](/apis/geocode/governmentdata-postzon-retrievebycoordinates-v1.2/)

#### GovernmentData Postzon RetrieveByPostcode (v1.5)

Returns the Postzon record for the given postcode.

[View the API](/apis/geocode/governmentdata-postzon-retrievebypostcode-v1.5/)

#### GovernmentData Postzon RetrieveByPostcodeOrPlace (v1)

Returns the Postzon record for the given postcode.

[View the API](/apis/geocode/governmentdata-postzon-retrievebypostcodeorplace-v1/)

#### StoreFinder Interactive FindPlaceNames (v1.1)

Retrieves the names of places.

[View the API](/apis/geocode/storefinder-interactive-findplacenames-v1.1/)

#### StoreFinder Interactive RetrieveCoordinates (v1.1)

Retrieves the coordinates of a place found using the FindPlaceNames service.

[View the API](/apis/geocode/storefinder-interactive-retrievecoordinates-v1.1/)
