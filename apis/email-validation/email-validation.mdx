#### EmailValidation Batch Validate (v1.2)

This batch processing service verifies up to 100 emails per batch.

[View the API](/apis/email-validation/batch-validate-v1.2/)

#### EmailValidation Interactive Validate (v2)

Designed for single email validation requests, this product verifies the
existence of an individual email address.

[View the API](/apis/email-validation\interactive-validate-v2/)

#### Bulk Email Validation

For very large batches, this asynchronous API allows you to submit requests
containing up to 100,000 emails.

[View the API](/apis/email-validation/bulk-email-validation/)
