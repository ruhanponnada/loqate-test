## Release Information

Please find the release notes and resolved case lists below for the current and all previous Loqate releases. If you are unsure about your API version, please [follow these steps](/release-information/how-to-obtain-version-information/).

## Release Schedule

Our data releases and related fixes are now done on a monthly frequency allowing our customers to get access to the most up to date global address data quicker. Customers can continue to download data using the API schedule below or can chose to get the latest data at the start of each month, by running the installation manager. This will automatically install any data for countries updated in the last month.

Release notes for data updates will be published below and these will be rolled up into the release notes for API releases so customers can clearly see what is in each release they chose to install.

Please note, release schedules can be subject to change.
